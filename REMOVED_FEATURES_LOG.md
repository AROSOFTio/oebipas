# Removed Features Log

The following proposal-external features were deleted from the codebase:

- Feedback controllers, routes, and pages
- Audit log controllers, routes, and service
- Service connection controllers and routes
- Meter controllers and routes
- Settings controllers and routes
- Extra user management routes for unsupported roles
- Global search page
- Admin-only dashboard and enterprise support pages
- Receipt-specific management screens
- Connection, meter, and support-ticket navigation items
- Database tables for feedback, audit logs, service connections, meters, settings, receipts, payment reconciliations, bill items, and user-role join tables

The resulting system now keeps only the modules justified by the approved proposal.
